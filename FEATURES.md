# 구현 기능 목록 (Gorail 프로토타입)

프로젝트 전체에서 **현재까지 구현·검증된 기능**을 부류별로 정리한 표. 갱신: 2026-06-04, 브랜치 `feature/chase-slice`.

- **근거 표기:** `§` = 추격 기획서(`chase_prototype_spec_v2_patched.docx`) / `scene§` = 씬 시스템 기획서(`scene_system_spec.docx`) / `D#` = 기획서 밖 추가 지시([EXTRA_DIRECTIVES.md](EXTRA_DIRECTIVES.md)) / `H#` = 편의 버퍼 백로그
- **상태:** ✅ 구현·Play 검증 / 🟡 구현·부분검증
- **비고:** 한 줄로 안 담기는 기능의 세부 묶음을 표기
